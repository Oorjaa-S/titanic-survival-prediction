# Titanic Survival Prediction

This project uses machine learning techniques to predict passenger survival on the Titanic dataset from Kaggle.

The project demonstrates a complete beginner-level machine learning workflow including:
- Exploratory Data Analysis (EDA)
- Missing Value Imputation
- Categorical Feature Encoding
- Feature Selection
- Train-Test Split
- Random Forest Classification
- Model Evaluation
- Kaggle Submission Workflow

The model was trained using a Random Forest Classifier and evaluated using accuracy metrics.

---

## Dataset

Dataset Source:  
Kaggle - Titanic: Machine Learning from Disaster

https://www.kaggle.com/competitions/titanic

### Training Dataset
Contains passenger information along with survival labels.

### Testing Dataset
Contains passenger information without survival labels for final prediction submission.
